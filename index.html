Claude Code

Anthropic’s agentic coding tool that can read, modify, and execute code in your working directory.

Install

Deploy the CLI tool to your local environment with this command:

Terminal / Shell
curl -fsSL https://claude.ai/install.sh | sh

Usage with Ollama

Open models are fully compatible through Ollama’s Anthropic-compliant API bridge.

qwen3.5 glm-5:cloud kimi-k2.5:cloud minimax-m2.7:cloud glm-4.7-flash

Quick Setup

Ollama CLI
ollama launch claude

Launch with specific model

Model Launch
ollama launch claude --model kimi-k2.5:cloud

Non-interactive (headless) mode

Run Claude Code in background processes, Docker, or CI/CD pipelines. The --yes flag skips all interactive prompts.

Headless Command
ollama launch claude --model kimi-k2.5:cloud --yes

Scheduled Tasks with /loop

Automate recurring research, bug reporting, or code reviews using intervals.

Syntax
/loop <interval> <prompt or /command>

Automation Examples

  • PR Reviews: /loop 30m Check my open PRs and summarize changes
  • Research: /loop 1h Research the latest AI news and report back
  • Bug Triage: /loop 15m Check for new GitHub issues
  • Reminders: /loop 1h Remind me to review the dependencies

Telegram & Plugins

Connect a bot via @BotFather to chat with Claude Code remotely. Launch with the channels flag:

Telegram Command
ollama launch claude -- --channels telegram
Pro Tip: For autonomous operation in isolated environments, use --dangerously-skip-permissions to bypass manual verification.

Manual Configuration

Set these environment variables to pair Claude Code manually with the Ollama local host:

Environment Export
export ANTHROPIC_AUTH_TOKEN=ollama export ANTHROPIC_API_KEY="" export ANTHROPIC_BASE_URL=http://localhost:11434

Run directly via Claude once configured:

Direct Exec
claude --model qwen3.5

System Requirements

Important: Claude Code requires a massive context window. It is recommended to configure Ollama for at least 64k tokens to ensure agentic stability.
