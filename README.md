# Claude Code: The Editorial Guide 📖

A beautifully crafted, single-page documentation site for **Anthropic's Claude Code**. This project provides a clean, literary-inspired interface to help developers set up agentic coding with local models via **Ollama**.

![Design Preview](https://img.shields.io/badge/Design-Editorial-D96A46)
![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Key Features
- **Claude-Inspired Aesthetic:** Warm cream backgrounds (`#FAF8F5`) and terracotta accents (`#D96A46`).
- **Premium Typography:** Sophisticated Lora Serif paired with clean Inter Sans-serif.
- **Robust Copy Engine:** One-click copy for terminal commands with built-in fallbacks for all browsers.
- **Full Documentation:** Includes Install, Ollama models, Headless mode, `/loop` scheduling, and Manual Setup.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.
- **Custom Loading Screen:** Featuring branding for **Istab Maibam**.

## 🚀 Quick Start
1. Clone this repository.
2. Open `index.html` in any web browser.
3. Use the terminal blocks to copy and paste setup commands directly into your CLI.

## 🛠 Tech Stack
- HTML5
- CSS3 (Custom Variables & Flexbox)
- JavaScript (ES6+ Clipboard API)
- Google Fonts (Lora, Inter, JetBrains Mono)
